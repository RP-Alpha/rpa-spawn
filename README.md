# rpa-spawn

A modern spawn selection system with Skycam transitions.

## Features
- **Spawn Selector**: NUI overlay to choose spawn location.
- **Skycam**: Cinematic camera transition from sky to player on spawn.
- **Last Location**: Support for spawning at last known location.

## Installation
1. Ensure `rpa-lib` is started.
2. Add `ensure rpa-spawn` to your `server.cfg`.

## Usage
Triggers automatically on player join/resource start (configurable).
